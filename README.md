E aí, cientista maluco!

Tá afim de analisar o teste do labirinto em Y nos seus amiguinhos roedores de um jeito mais tecnológico? Então esse repositório é pra você!

Desenvolvi uma aplicaçãozinha web bem bacana pra te ajudar a registrar as sequências percorridas pelos bichinhos e calcular o índice de alternância.

Como usar?

Dá um clone nesse repositório: git clone https://github.com/seu_usuario/labirinto_em_Y.git (troca "seu_usuario" pelo teu username no GitHub )
Manda ver o código: Abre o index.html e script.js pra ver a mágica acontecendo.
Solta os roedores! Registra as entradas deles no labirinto (A, B ou C) clicando nos botões correspondentes.
Analisa o resultado: A aplicação vai te mostrar a sequência percorrida, o número de entradas e o índice de alternância.
O que é o índice de alternância?

É uma medida pra saber se o roedor tá explorando o labirinto de forma aleatória, visitando os braços de forma alternada, ou se ele tem uma preferência por um lado específico.

Dá um reset!

Quando quiser começar uma nova bateria de testes, aperta o botão "Resetar".

Contribua!

Esse código tá aberto pra melhorias! Se quiser adicionar funcionalidades ou deixar a interface mais bonita, sinta-se à vontade pra fazer um pull request.

Compartilhe o conhecimento!

Esse projeto é open source, então compartilha com outros cientistas malucos que tão estudando a memória dos roedores!

Bora brincar de ciência!
